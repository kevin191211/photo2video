# 照片轉影片工具 (photo2video)

用 Rust + egui 寫的桌面小工具，把一連串照片合成一支影片。

## 使用方式

執行本資料夾中的 `photo2video.exe`（請與 `ffmpeg.exe` 放在同一資料夾），操作步驟：

1. **加入照片**：點「選擇資料夾」或「選擇照片（可多選）」，也可以直接把照片或資料夾拖曳進視窗，或對照片按右鍵用「開啟方式」選擇本程式。照片會依檔名自然排序（img2 會排在 img10 前面），在下方縮圖上按右鍵可移除單張或清空全部，← → 方向鍵可切換預覽。
2. **設定每秒張數 (fps)**：1～60，預設 10，例如 2 代表每秒播 2 張照片；修改後會自動記住，下次開啟直接套用。
3. **調色**：右側面板可調整色溫、色調、曝光度、對比、亮度、陰影、白色、黑色、清晰度、鮮豔度、飽和度（各 -100～+100，滑桿連點兩下可歸零）。點選下方任一張縮圖即可在中央大預覽區即時檢視效果；預覽與輸出使用同一套 FFmpeg 濾鏡，所見即所得。預設參數套用到影片中的**每一張照片**；想只調特定照片，可按住 **Ctrl 點縮圖**多選（Shift+點可連選一段），此時滑桿只影響選取的照片，有個別調色的縮圖右下角會標示 🎨，也可一鍵「取消個別調色」改回跟隨全部。
4. **文字（依段落設定）**：按右側「文字」區的「＋ 新增文字」，設定「從第 X 張到第 Y 張」與要顯示的文字（可多行），這段照片就會顯示同一段文字；可新增多段，適合上百張照片的場合。文字可直接在中央預覽區**拖曳移動**到任意位置，點選後可用角落把手縮放、頂端圓形把手旋轉（靠近 45° 的倍數會自動吸附），大小與旋轉角也可在右側用滑桿調整；字型（微軟正黑體、標楷體、Arial 等系統字型）、文字顏色、外框粗細/顏色、半透明底框為全部文字共用；有文字的照片縮圖右上角會標示 💬，點選縮圖可預覽該張的實際效果。
5. **轉場、動態縮放與背景音樂**：「轉場與音樂」區塊可選「淡入淡出」轉場、勾選 Ken Burns 動態縮放（照片緩慢推近/拉遠，啟用任一者輸出會提升為 30fps）；背景音樂支援 MP3 / WAV / M4A / FLAC / OGG（也可直接拖進視窗），可調音量與結尾淡出，音樂比影片短會自動循環、比影片長會自動裁切。
6. **選擇輸出格式**：MP4 / MKV / MOV / AVI（H.264）或 WebM（VP9），並可選解析度（720p～4K，或「原始像素」依照片中最大寬高輸出）。照片會等比例縮放置中，不足處補黑邊（調色在補黑邊之前套用，黑邊不會被調亮）。轉換會自動偵測並使用硬體編碼（NVENC / QSV / AMF），沒有可用 GPU 時退回 libx264。
7. **儲存專案 / 開啟舊檔**：工具列的「💾 儲存專案」（快捷鍵 Ctrl+S）會把照片清單與所有設定（每秒張數、解析度、輸出格式、調色含個別照片調色、字幕、轉場、Ken Burns、背景音樂）存成一個 `.p2v` 專案檔；下次點「📂 開啟專案」（Ctrl+O）、直接把專案檔拖進視窗，或在啟動畫面的「**最近的專案**」清單點檔名，就能還原上次的工作，不用重新選資料夾、重新設定。「🆕 新專案」則會清空照片與所有設定重新開始。專案檔只記錄照片的路徑，照片本身仍在原資料夾，若照片被移動或刪除，開啟時會提示並略過該張。
8. 按右下角「**▶ 開始轉換**」，選擇儲存位置後開始轉換，底部會顯示進度；轉換途中可按進度旁的「**✕ 取消**」隨時中止（會自動刪除未完成的影片，照片與設定都保留）。轉換完成後可按「**▶ 播放**」直接以預設播放器預覽成品，或「**開啟資料夾**」在檔案總管中選取它。右側各設定區塊的標題可點擊收合。

> 本資料夾已附上 `ffmpeg.exe`，不需另外下載；若缺少該檔案，第一次轉換時會自動下載（約 80MB）。

## 問題回報

- **轉換失敗**時，錯誤訊息旁有「🔗 回報問題」按鈕，點了會開啟回報頁面，並自動帶入錯誤內容與版本，送出即完成回報。
- **程式意外關閉（閃退）**時，下次開啟程式會在視窗底部顯示「⚠ 程式上次異常關閉」通知，同樣點「🔗 回報問題」即可回報。

## 命令列模式

也可以不開視窗直接轉換：

```
photo2video.exe --cli <照片資料夾> <fps> <輸出檔>
```

例如：`photo2video.exe --cli D:\photos 2 D:\out.mp4`
（輸出格式由副檔名決定：mp4 / mkv / mov / avi / webm）

## 支援的照片格式

jpg / jpeg / png / bmp / webp / tif / tiff

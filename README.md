# 照片轉影片工具 (photo2video)

用 Rust + egui 寫的桌面小工具，把一連串照片合成一支影片。

## 下載

不想自己建置的話：

- **最新版直接下載**：<https://github.com/kevin191211/photo2video/releases/latest/download/photo2video-windows-x64.zip>（永遠指向最新版）
- [Releases 頁面](https://github.com/kevin191211/photo2video/releases/latest) 可看歷史版本與更新紀錄

zip 內含 `photo2video.exe`、`ffmpeg.exe` 與使用說明，解壓即可使用。程式啟動時會自動檢查更新並在視窗底部提示。

## 使用方式

執行 `target\release\photo2video.exe`，操作步驟：

1. **加入照片**：點「選擇資料夾」或「選擇照片（可多選）」，也可以直接把照片或資料夾拖曳進視窗，或在啟動時把照片／資料夾當參數帶入（支援「開啟方式」）。照片會依檔名自然排序（img2 會排在 img10 前面），在下方縮圖上按右鍵可移除單張或清空全部，← → 方向鍵可切換預覽。
2. **設定每秒張數 (fps)**：1～60，預設 10，例如 2 代表每秒播 2 張照片；修改後會自動記住，下次開啟直接套用。
3. **調色（套用至整部影片）**：右側面板可調整色溫、色調、曝光度、對比、亮度、陰影、白色、黑色、清晰度、鮮豔度、飽和度（各 -100～+100）。點選下方任一張縮圖即可在中央大預覽區即時檢視調色效果；預覽與輸出使用同一套 FFmpeg 濾鏡，所見即所得。轉換時參數會套用到影片中的**每一張照片**。
4. **字幕（依段落設定）**：按字幕區右上的「＋ 新增段落」，設定「從第 X 張到第 Y 張」與要顯示的文字（可多行），這段照片就會共用同一句字幕；可新增多個段落，適合上百張照片的場合。字型（微軟正黑體、標楷體、Arial 等系統字型）、大小、文字顏色、外框粗細/顏色、位置（上/中/下）、半透明底框皆可調整，樣式全域共用；有字幕的照片縮圖右上角會標示 💬，點選縮圖可預覽該張的實際效果。
5. **轉場、動態縮放與背景音樂**：「轉場與音樂」區塊可選「淡入淡出」轉場、勾選 Ken Burns 動態縮放（照片緩慢推近/拉遠，啟用任一者輸出會提升為 30fps）；背景音樂支援 MP3 / WAV / M4A / FLAC / OGG（也可直接拖進視窗），可調音量與結尾淡出，音樂比影片短會自動循環、比影片長會自動裁切。
6. **選擇輸出格式**：MP4 / MKV / MOV / AVI（H.264）或 WebM（VP9），並可選解析度（720p～4K）。照片會等比例縮放置中，不足處補黑邊（調色在補黑邊之前套用，黑邊不會被調亮）。轉換會自動偵測並使用硬體編碼（NVENC / QSV / AMF），沒有可用 GPU 時退回 libx264。
7. 按「**開始轉換**」，選擇儲存位置後開始轉換，會顯示進度條。右側各設定區塊的標題可點擊收合。

> 第一次轉換時會自動下載 FFmpeg（約 80MB），之後不需再下載。

## 命令列模式

也可以不開視窗直接轉換：

```
photo2video.exe --cli <照片資料夾> <fps> <輸出檔>
```

例如：`photo2video.exe --cli D:\photos 2 D:\out.mp4`
（輸出格式由副檔名決定：mp4 / mkv / mov / avi / webm）

## 建置

```
cargo build --release
```

## 技術說明

- GUI：eframe / egui（深色剪輯風格介面，自動載入微軟正黑體顯示繁體中文）
- 轉檔：ffmpeg-sidecar（自動下載並呼叫 FFmpeg，用 concat demuxer 串接照片）
- 支援的照片格式：jpg / jpeg / png / bmp / webp / tif / tiff
